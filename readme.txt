== Installation Guide ==
1.    Go to WordPress Control Panel
2.    Click "Plugins", then "Add New"
3.    Type in "Visual Editor" and click "Search Plugins"
4.    Download and install the Plugin
5.    Click the "Activate Plugin" link
6.    Move to the "Visual Editor" section in the menu
7.    Click "Begin Website Editing"
8.    You will be redirected to the editor. Done!
